# Creative Studio Multi-Page Template

## Contexte

Ce projet vise à transformer le template **Creative Studio One-Page** en un modèle **Multi-Page** pour améliorer l'expérience utilisateur.

## Exigences

- Transformation Multi-Page : Les rubriques "Services", "Portfolio", "Blog", et "Contact" doivent mener à des pages distinctes.
- Responsive Design: Adapté pour toutes les plateformes (web, mobile, desktop).
- Formulaire de Contact: Ajout de deux nouveaux champs d'entrée.
- Modifications de la Page d'Accueil** : Respecter le thème original tout en apportant des améliorations.
- Bonnes Pratiques HTML/CSS : Code minimal et optimisé.
- Logo: Création d'un logo en accord avec le thème.
- Améliorations Créatives** : Suggestions d'améliorations supplémentaires.

## Bonus
1. J'ai ajouté un bouton pour diviser la page en quatre petites pages.
2. Ajout d'une nouvelle page (insertion).
3. Flexibilité dans la navigation entre les pages.
4. Changement du logo de la page.
5. Changement de l'arrière-plan de la page.
6. Ajout d'un nouvel input sur la page de contact.
7. Maintien de la réactivité de la page.
8. J'ai ajouté des améliorations au fichier CSS.
